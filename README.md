## 주제

- 함께 만드는 알고리즘 저지 사이트

## 구현 기능

- 유저
    - 회원가입 기능
    - 로그인 기능
    - 로그아웃 기능
    - 회원 탈퇴 기능
- 알고리즘 문제
    - 검색 기능
    - 제작 기능
    - 업데이트 기능
    - 코드 제출 기능
    - 코드 채점 기능
    - 게시판, 게시글, 댓글
- 관리자 기능
- 레벨 시스템

## 브랜치
- main : 배포 버전
- develop : 개발 버전
- feature : 기능 개발
- 로컬에서 develop 브랜치에서 분기한 feature 브랜치에서 작업을 진행
- 기능 개발이 끝나면 develop로 merge
- 매주 화요일 main으로 merge
