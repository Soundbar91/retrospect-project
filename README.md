## 구현 기능

- 유저
    - 회원가입 기능
    - 로그인 기능
    - 로그아웃 기능
    - 회원 탈퇴 기능
- 알고리즘 문제
    - 검색 기능
    - 제작 기능
    - 업데이트 기능
    - 코드 제출 기능
    - 코드 채점 기능
    - 게시판, 게시글, 댓글
- 레벨 시스템

## 개발환경
- IDE : IntelliJ IDEA
- SDK : corretto-17
- DB : MySQL Workbench 8.0 CE
- OS : Windows 10, Ubuntu 22.04.4 LTS
